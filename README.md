
# Ivan’s Barbershop — Web Project

Proyecto web con Vite + React + TypeScript + Tailwind + shadcn-ui.

## Scripts
```bash
npm i
npm run dev
npm run build
npm run preview
```

## Deploy rápido
- **Netlify**: sube la carpeta `dist/` a https://app.netlify.com/drop
- **Vercel**: conecta el repo y despliega con autodetección (Vite)
- **Hosting propio**: sube el contenido de `dist/` al raíz del sitio
